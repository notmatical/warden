The workflow editor's side peek panel is gone — node config is now edited inline on the card, Unreal-blueprint style. Each card shows its label, intent, model, effort, prompt, and branch by default, height-capped with a gradient fade and a ghost chevron that expands to the full height when there's more (run status, errors via Callout, and an output preview); the full transcript opens the session on double-click. Gate nodes show their status, approval controls, and explainer inline too.
